This is an implementation of the game Memory as an applet by using JDK 1.1.8. All source code was developed during the project Erinnerung in the end of 2001.

The applet has a fixed size of 720 x 502 pixel. All code is included in the archive erinnerung.jar, main class is de.a46.memory.Starter.class. The code has been optimized for IE 5.5, Netscape 4.7 and 6.1. There are several parameters, all of them required:

SOUND_FLAG 'true' or 'false'. Indicates, whether sound files should be played when turning cards.
DELAY_FLAG 'true' or 'false'. Indicates, whether small pauses should be included when loading data. This is useful when not playing via internet -- the welcome image keeps shown about 10 seconds.
MULTIMEDIAS_FILE codebase-relative path to the multimedias file, containing paths to all card images and audios.
PROPERTIES_FILE codebase-relative path to the properties file, containing paths to all other images.
AUTOCLOSE_FILE codebase-relative path to a html-file, that is called when the user exites the program by clicking the exit-button. Originally this file just contains a 'self.close()' javascript command.
HELPTEXT_FILE codebase-relative path to a textfile containing lines of text to get shown, when the user clicks the help button. Text must be formatted line by line to fit to the HelpPanel, a blank line indicates a period.
IMAGE_BASE subdirectory containing all card images as they are listed in the multimedias file.
AUDIO_BASE subdirectory containing all card audios as they are listed in the multimedias file.
DATA_BASE subdirectory containing all layout images as they are listed in the properties file.
BACKGROUND_BASE subdirectory containing all background images as listed in the properties file.

The code consists of two packages.

  • de.a46.memory contains functionality to initialize the program, to load external resources and to perform the moves in the game.
  • de.a46.memory.gui contains all gui functionality. All components are structured as a tree, where class MainPanel is the root node.
