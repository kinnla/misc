Package de.a46.memory includes all classes containing technical functionality for the memory game. Main class is Starter which initializes the program. Also essential are class Loader, which reads all external resources, and class Game, which acts like a final state machine and controls the moves in the game.

The connection between the layer represented by this package and the gui layer is made by two interfaces and one reference.

  • Interface de.a46.memory.gui.RessourceContainer (implemented by class Loader) defines methods to provide images, audioclips and text, as needed by differnend gui-classes, but also by class Game (audioclips) and class Starter (welcomeImage).
  • Interface de.a46.memory.UserActionListener defines methods representing all possible user actions that can happen to the gui and must be responsed by members of this package.
  • In class Starter, that is responsible to initialize all components, there is a direct reference to class de.a46.memory.gui.MainPanel.
