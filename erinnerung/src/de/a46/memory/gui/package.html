Package de.a46.memory.gui contains all gui functionality. The components are structured as a tree, where class MainPanel is the root component, representing the whole plane of 720 x 502 pixel. There are three child components implemented by class ButtonbarPanel, by class LeftPanel and by abstract class RightPanel. In correspondence to user action a concrete instance of class FieldPanel, class HelpPanel or class ExitPanel is shown, all subclassing RightPanel. All resources the gui needs are defined in interface RessourceContainer.

Also a few basic components are defined. Abstract class MPanel defines a component that casts events when the mouse enters, clicks or exits. Its subclass MButton acts like a common java.awt.Button, but displayes images representing eg. the pressed-state or the disabled-state. MPanel is also subclassed by MultiImagePanel, that displays one image from an array.
